# 💫 About Me:

Code. Build. Scale. Innovate.
I’m a Second-year CSE (AI/ML) student who enjoys building things—especially where AI meets real-world problems. Lately, I’ve been exploring how intelligent systems and cloud tools can actually be used to create something useful, not just theoretical.
During my 6-week AI & ML internship (AICTE + Edunet + IBM), I worked on TripTuner, an AI-based student travel planner chatbot. That’s where I really got hands-on with Generative AI, prompt design, and connecting everything into a working system.
I’ve been learning and experimenting with:

 Google Cloud (BigQuery ML, Cloud Run, GKE, CI/CD basics) 
 Python, C++, and DSA 
I’m not just interested in building models—I like working on complete systems that actually run, scale, and solve something meaningful.
Outside of academics, I take part in hackathons and coding challenges to push myself and learn faster. Still figuring things out, improving step by step, and trying to build better projects each time.
Going forward, my focus is to build strong expertise in AI + Cloud and work on production-level systems—especially in Generative AI and scalable backend architectures. I want to contribute to real-world products where technology creates actual impact, not just demos.
Right now, I’m looking for opportunities in AI/ML, cloud, or development roles where I can learn, contribute, and grow through real work
